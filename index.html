Tareas de Codigo Estructura de Datos y Análisis

Tarea 1: Implementar Algoritmo y Pseudocódigo

En esta tarea se desarroló un Diagrama de Flujo y su correspondiente pseudocódigo para crear un algoritmo capaz de calcular el promedio, la suma y el máximo de un conjunto variable de números.

Tarea2: Algoritmos de Ordenamiento

En esta tarea debía realizar una función que genera 10,000 números aleatorios enteros y se guardaban en un array, con números entre 0 y 10,000. Luego, utilizar este array para comparar diferentes algoritmos de ordenamiento en JavaScript y determinar cuál era el método más rápido. Al final, se mostraron los resultados de todos los algoritmos y se dio un veredicto. Los métodos de ordenamiento que se usaron son Bubble Sort, Selection Sort, Insertion Sort, Merge Sort y Quick Sort.

Tarea 3: Aplicación Lista de Tareas

Se creó una aplicación de lista de tareas que permitía agregar, marcar como completadas y eliminar tareas usando una lista enlazada, con puntos extras si era doblemente enlazada. Las tareas se añadían con un campo de entrada y un botón, se marcaban con una casilla de verificación y se eliminaban con un botón. Las tareas persistían después de recargar la página usando almacenamiento local. Se usaron HTML y CSS para la interfaz, JavaScript para la lógica, y un repositorio de Github con Github Pages para alojar la aplicación.
